# Dead Simple Papers

VIT Previous Year Papers without the noise. fast, lightweight, and no-nonsense.
<img width="1408" height="900" alt="image" src="https://github.com/user-attachments/assets/e8430d30-a058-41a7-b56a-53ebdec8166b" />

## Stack

- **Frontend**: Vanilla TypeScript + CSS
- **Backend**: Go

## Development

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
go run main.go
```
